2. Logistic regression model¶

The objective function is $$Q (w; X, y) = \frac{1}{n} \sum_{i=1}^n \log \Big( 1 + \exp \big( - y_i x_i^T w \big) \Big) + \frac{\lambda}{2} \| w \|_2^2 .$$

In [6]:
# Calculate the objective function value
# Inputs:
#     w: d-by-1 matrix
#     x: n-by-d matrix
#     y: n-by-1 matrix
#     lam: scalar, the regularization parameter
# Return:
#     objective function value (scalar)
def objective(w, x, y, lam):
    n, d = x.shape
    yx = numpy.multiply(y, x)        # n-by-d matrix
    yxw = numpy.dot(yx, w)           # n-by-1 matrix
    vec1 = numpy.exp(-yxw)           # n-by-1 matrix
    vec2 = numpy.log(1 + vec1)       # n-by-1 matrix
    loss = numpy.mean(vec2)          # scalar
    reg = lam / 2 * numpy.sum(w * w) # scalar
    return loss + reg

3. Numerical optimization¶

3.1. Gradient descent¶

The full gradient at $w$ is $$g = \nabla Q(w) = - \frac{1}{n} \sum_{i=1}^n \frac{y_i x_i}{1 + \exp(y_i x_i^T w)} + \lambda w.$$

In [8]:
# Calculate the gradient
# Inputs:
#     w: d-by-1 matrix
#     x: n-by-d matrix
#     y: n-by-1 matrix
#     lam: scalar, the regularization parameter
# Return:
#     g: d-by-1 matrix, full gradient
def gradient(w, x, y, lam):
    n, d = x.shape
    yx = numpy.multiply(y, x)                          # n-by-d matrix
    yxw = numpy.dot(yx, w)                             # n-by-1 matrix
    vec1 = numpy.exp(yxw)                              # n-by-1 matrix
    vec2 = numpy.divide(yx, 1 + vec1)                  # n-by-d matrix
    vec3 = -numpy.mean(vec2, axis=0).reshape(d, 1)     # d-by-1 matrix
    g = vec3 + lam * w
    return g
In [9]:
# Gradient descent for solving logistic regression
# Inputs:
#     x: n-by-d matrix
#     y: n-by-1 matrix
#     lam: scalar, the regularization parameter
#     stepsize: scalar
#     max_iter: integer, the maximal iterations
#     w: d-by-1 matrix, initialization of w
# Return:
#     w: d-by-1 matrix, the solution
#     objvals: a record of each iteration's objective value
def grad_descent(x, y, lam, stepsize, max_iter=100, w=None):
    n, d = x.shape
    objvals = numpy.zeros(max_iter)   # store the objective values
    if w is None:
        w = numpy.zeros((d, 1))       # zero initialization

    for t in range(max_iter):
        objval = objective(w, x, y, lam)
        objvals[t] = objval
        g = gradient(w, x, y, lam)
        w = w - stepsize * g

    print('Final objective value of GD: ' + str(objvals[-1]))
    return w, objvals

Run gradient descent. For GD, one iteration is one epoch, so max_iter = 100 means 100 epochs.

3.2. Stochastic gradient descent (SGD)¶

Define $Q_i (w) = \log \Big( 1 + \exp \big( - y_i x_i^T w \big) \Big) + \frac{\lambda}{2} \| w \|_2^2 $.

The stochastic gradient at $w$ is $g_i = \frac{\partial Q_i}{\partial w} = -\frac{y_i x_i}{1 + \exp(y_i x_i^T w)} + \lambda w.$

In [11]:
# Calculate the objective Q_i and the gradient of Q_i
# Inputs:
#     w: d-by-1 matrix
#     xi: 1-by-d matrix
#     yi: scalar
#     lam: scalar, the regularization parameter
# Return:
#     obj: scalar, the objective Q_i
#     g: d-by-1 matrix, gradient of Q_i
def stochastic_objective_gradient(w, xi, yi, lam):
    xi = numpy.asarray(xi).reshape(1, -1)   # ensure xi is a 1-by-d row
    yx = yi * xi                            # 1-by-d matrix
    yxw = float(numpy.dot(yx, w).item())    # scalar

    # calculate objective function Q_i
    loss = numpy.log(1 + numpy.exp(-yxw))   # scalar
    reg = lam / 2 * numpy.sum(w * w)        # scalar
    obj = loss + reg

    # calculate stochastic gradient
    g_loss = -yx.T / (1 + numpy.exp(yxw))   # d-by-1 matrix
    g = g_loss + lam * w                    # d-by-1 matrix

    return obj, g
In [12]:
# SGD for solving logistic regression
# Inputs:
#     x: n-by-d matrix
#     y: n-by-1 matrix
#     lam: scalar, the regularization parameter
#     stepsize: scalar
#     max_epoch: integer, the maximal epochs
#     w: d-by-1 matrix, initialization of w
# Return:
#     w: the solution
#     objvals: record of each epoch's average objective value
def sgd(x, y, lam, stepsize, max_epoch=100, w=None):
    n, d = x.shape
    objvals = numpy.zeros(max_epoch)   # store the objective values
    if w is None:
        w = numpy.zeros((d, 1))        # zero initialization

    for t in range(max_epoch):
        # randomly shuffle the samples
        rand_indices = numpy.random.permutation(n)
        x_rand = x[rand_indices, :]
        y_rand = y[rand_indices, :]

        objval = 0   # accumulate the objective values
        for i in range(n):
            xi = x_rand[i, :]          # 1-by-d matrix
            yi = float(y_rand[i, 0])   # scalar
            obj, g = stochastic_objective_gradient(w, xi, yi, lam)
            objval += obj
            w = w - stepsize * g

        stepsize *= 0.9   # decrease step size to guarantee convergence
        objval /= n
        objvals[t] = objval

    print('Final objective value of SGD: ' + str(objvals[-1]))
    return w, objvals

Run SGD. For SGD, one epoch visits all $n$ samples once (i.e., $n$ iterations).

6. Mini-batch SGD (the implementation task)¶

6.1. Compute the objective $Q_I$ and its gradient using a batch of samples¶

Define $$Q_I (w) = \frac{1}{b} \sum_{i \in I} \log \Big( 1 + \exp \big( - y_i x_i^T w \big) \Big) + \frac{\lambda}{2} \| w \|_2^2 ,$$ where $I$ is a set of $b$ indices drawn from $\{1, \cdots, n\}$ without replacement.

The mini-batch gradient at $w$ is $$g_I = \frac{\partial Q_I}{\partial w} = \frac{1}{b} \sum_{i \in I} \frac{- y_i x_i}{1 + \exp(y_i x_i^T w)} + \lambda w.$$

Everything below is computed with matrix-vector multiplication over the whole batch at once (no inner FOR loop over the $b$ samples).

In [18]:
# Calculate the objective Q_I and the gradient of Q_I
# Inputs:
#     w:   d-by-1 matrix
#     xi:  b-by-d matrix (a batch of b feature vectors)
#     yi:  b-by-1 matrix (the corresponding labels)
#     lam: scalar, the regularization parameter
#     b:   integer, the batch size
# Return:
#     obj: scalar, the objective Q_I
#     g:   d-by-1 matrix, gradient of Q_I
def mb_stochastic_objective_gradient(w, xi, yi, lam, b):
    # yx[i, :] = yi * xi  (broadcast the label across each feature row)
    yx = numpy.multiply(yi, xi)            # b-by-d matrix
    yxw = numpy.dot(yx, w)                 # b-by-1 matrix  (matrix-vector product)

    # calculate the objective function Q_I (averaged over the batch)
    loss = numpy.mean(numpy.log(1 + numpy.exp(-yxw)))   # scalar
    reg = lam / 2 * numpy.sum(w * w)                    # scalar
    obj = loss + reg

    # calculate the mini-batch gradient (averaged over the batch)
    d = w.shape[0]
    vec = numpy.divide(yx, 1 + numpy.exp(yxw))          # b-by-d matrix
    g_loss = -numpy.mean(vec, axis=0).reshape(d, 1)     # d-by-1 matrix
    g = g_loss + lam * w                                # d-by-1 matrix

    return obj, g

6.2. Implement mini-batch SGD¶

Following the SGD implementation:

  1. In every epoch, randomly permute the $n$ samples.
  2. Each epoch has $\frac{n}{b}$ iterations; each iteration uses the next $b$ samples to compute the objective and gradient via mb_stochastic_objective_gradient.
  3. Record one objective value per epoch (the mean of the batch objectives).
  4. Decrease the step size after every epoch (as in SGD).
In [19]:
# Mini-Batch SGD for solving logistic regression
# Inputs:
#     x: n-by-d matrix
#     y: n-by-1 matrix
#     lam: scalar, the regularization parameter
#     b: integer, the batch size
#     stepsize: scalar
#     max_epoch: integer, the maximal epochs
#     w: d-by-1 matrix, initialization of w
# Return:
#     w: the solution
#     objvals: record of each epoch's average objective value
def mb_sgd(x, y, lam, b, stepsize, max_epoch=100, w=None):
    n, d = x.shape
    objvals = numpy.zeros(max_epoch)    # store one objective value per epoch
    if w is None:
        w = numpy.zeros((d, 1))         # zero initialization

    num_batch = int(numpy.floor(n / b)) # number of iterations per epoch

    for t in range(max_epoch):
        # randomly shuffle the samples (just like SGD)
        rand_indices = numpy.random.permutation(n)
        x_rand = x[rand_indices, :]
        y_rand = y[rand_indices, :]

        objval = 0   # accumulate the per-batch objective values
        for j in range(num_batch):
            # take the j-th batch of b samples
            xi = x_rand[j*b : (j+1)*b, :]   # b-by-d matrix
            yi = y_rand[j*b : (j+1)*b, :]   # b-by-1 matrix
            obj, g = mb_stochastic_objective_gradient(w, xi, yi, lam, b)
            objval += obj
            w = w - stepsize * g

        stepsize *= 0.9          # decrease step size to guarantee convergence
        objval /= num_batch      # average objective over the batches in this epoch
        objvals[t] = objval

    print('Final objective value of MB-SGD (b=' + str(b) + '): ' + str(objvals[-1]))
    return w, objvals

8. Summary¶

  • Logistic regression with $\ell_2$ regularization was trained on the Diabetes dataset (640 train / 128 test samples, 8 standardized features plus a bias term).
  • Four optimizers were implemented and compared. As expected for a strongly convex objective, all four converge to the same minimum (objective $\approx 0.47$).
  • When plotted against epochs, SGD and MB-SGD reach a good solution faster than GD in the early epochs, because they update w many times per epoch instead of once.
  • The mini-batch objective and gradient are computed with matrix-vector multiplication over the whole batch, avoiding any inner per-sample FOR loop.
